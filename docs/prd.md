# 需求文档

## 1. 应用概述

- **应用名称**：Broken of Love 链接导航页
- **应用描述**：一个单页网页，包含个人头像与标题展示，以及小红书、微博、抖音、B站四个平台的链接群，每个平台链接群支持收拢与展开操作。

---

## 2. 页面结构与核心功能

### 2.1 页面总览

```
单页网页
├── 模块一：个人信息展示区
├── 模块二：小红书链接群
├── 模块三：微博链接群
├── 模块四：抖音链接群
└── 模块五：B站链接群
```

### 2.2 模块一：个人信息展示区

- 展示一个圆形头像图片，使用占位图代替（待用户后续替换）
- 头像下方展示标题文字：Broken of Love
- 布局居中对齐

### 2.3 模块二：小红书链接群

- 模块标题：小红书
- 支持点击标题区域进行收拢 / 展开切换，默认展开
- 展开状态下，展示若干条链接列表，每条链接包含：
  - 链接标题（描述文字）
  - 可点击跳转的链接地址
- 链接内容使用占位示例数据，待用户后续替换

### 2.4 模块三：微博链接群

- 模块标题：微博
- 支持点击标题区域进行收拢 / 展开切换，默认展开
- 展开状态下，展示链接列表，排版规则为**每行排列三个子模块**
- 已移除子模块：「微博日常更新」、「碎碎念合集」
- 每条链接包含：
  - 链接标题（描述文字）
  - 可点击跳转的链接地址
- 链接内容使用占位示例数据（已移除上述两项后的剩余内容），待用户后续替换

### 2.5 模块四：抖音链接群

- 模块标题：抖音
- 支持点击标题区域进行收拢 / 展开切换，默认展开
- 已移除子模块：「抖音短视频」、「直播预告」
- 新增以下三个子模块（链接标题从视频标题中提取前五个字，多余字符以省略号代替）：
  - 子模块1：标题从视频标题前五字提取（省略号补全），链接：https://www.douyin.com/video/7621157841429480163
  - 子模块2：标题从视频标题前五字提取（省略号补全），链接：https://www.douyin.com/video/7621157419745423055
  - 子模块3：标题从视频标题前五字提取（省略号补全），链接：https://www.douyin.com/video/7621156459053761910
- 每条链接点击后在新标签页中打开目标地址

### 2.6 模块五：B站链接群

- 模块标题：B站
- 支持点击标题区域进行收拢 / 展开切换，默认展开
- 展开状态下，展示若干条链接列表，每条链接包含：
  - 链接标题（描述文字）
  - 可点击跳转的链接地址
- 其中原子模块「《破碎的爱》播放平台」已更名为「Fabel油管海外平台」
- 其余链接内容使用占位示例数据，待用户后续替换

---

## 3. 业务规则与逻辑

- 每个平台链接群默认为展开状态
- 点击模块标题区域可切换收拢 / 展开状态，收拢时隐藏链接列表，仅显示模块标题
- 收拢 / 展开状态切换时，应有明显的视觉指示（如箭头图标方向变化）
- 链接点击后在新标签页中打开目标地址
- 微博链接群的子模块排版为每行三个，横向排列

---

## 4. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 头像图片加载失败 | 显示默认占位图或灰色圆形背景 |
| 链接地址为空 | 该条链接不可点击，视觉上置灰处理 |
| 链接群内无链接条目 | 展开后显示「暂无内容」提示文字 |

---

## 5. 验收标准

- 页面包含五个模块，顺序与需求一致
- 模块一正确展示占位头像与标题「Broken of Love」
- 四个平台链接群均可正常收拢与展开，状态切换有视觉反馈
- 链接点击后在新标签页中打开
- 微博链接群子模块以每行三个的方式横向排列展示
- 抖音链接群已移除「抖音短视频」与「直播预告」，新增三条抖音视频链接，标题格式为视频标题前五字加省略号
- B站链接群中原「《破碎的爱》播放平台」子模块已更名为「Fabel油管海外平台」
- 页面整体布局在桌面端正常显示
